Common CSS Properties

CSS allows us to Style our Documents in a way that makes them pleasing to the eye and organizes the content making it easy to interpret. There are a few common properties which are used almost 100% of the time when styling HTML documents. These properties are a simple but effective way to bring life to the content. With good CSS styling, a very basic document is tranformed into a beautiful presentation of information.

CSS Properties
Property Accepted Values Function
“color” keywords, hexadecimal, rgb/a, hsl/a adds color to foreground elements
“font-style” keywords determines typeface of letters
“font-size” keywords, pixels(px), percentages sets size of letters on the page
“font-weight” numerical value sets the thicknes of the selected font-family
“background-color” keywords, hexadecimal, rgb/a, hsl/a adds color to background elements
“background” keywords, gradient, images, percentages adds pictures and styling of color to background elements
“height/ width” keywords, pixels(px), percentages, em sets height and width of elements
“border” keywords sets style of element's visible boundaries
“border-style” style keywords defines the style of a border
“border-collapse” keywords sets borders as collapsed or separated
“text-transform” keywords specifies capitalization in an elemen's text
“margin” direction keywords, length, percentages sets spacing top/bottom/left/right of target element
“padding” keywords, pixels(px) sets spacing around element's content

These are just a few examples of CSS properties commonly used throughout stylesheets to give character to HTML documents.
